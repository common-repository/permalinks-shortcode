Permalink shortcode

== Description ==

With this plugin, once your shortcode is integrated inside your page, you can change the permalink's destination.
Once you have integrated the shortcode in your page, you can change the url of destination (permalink), your url will not be broken.
No more 404 error !!
To help you integrate the shorcode into your pages, you can use our shortcode generator.
Once it generated, you just have to copy / paste to the desired place in your page.
You can also embed the URLs of your tags and categories.


== Installation ==

This section describes how to install the plugin and get it working.

1.	Upload the «plksc» folder to the /wp-content/plugins/ directory
2.	Activate the plugin through the 'Plugins' menu in WordPress
3.	Go to Dashboard > Settings
